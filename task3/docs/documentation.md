# Library Management System

## Overview
Console-based Library Management System written in Go.

## Features
- Add Book
- Remove Book
- Borrow Book
- Return Book
- List Available Books
- List Borrowed Books

## Concepts Used
- Structs
- Interfaces
- Maps
- Slices
- Methods
- Modular Architecture

## Architecture
Controller → Service → Model

## Run Project
go run main.go
